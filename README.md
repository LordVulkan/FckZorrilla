# FckZorrilla

This is a userscript that hides all Mikel Zorrilla's posts from Espinof.com.

Just add https://github.com/LordVulkan/FckZorrilla/releases/latest/download/FckZorrilla.js to your user script manager.

## Development

``` sh
$ yarn dev
```

## Building

```sh
$ yarn build
```

## Lint

``` sh
$ yarn lint
```
